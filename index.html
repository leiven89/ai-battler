Original Character Relationship Battle Prototype

OC BATTLE LINK

戦いは、関係を生む。

キャラの勝敗だけでなく、台詞、記憶、感情、関係が残る オリキャラバトル試作です。今回は Command R+ による演出生成、 詳細キャラシート、備考欄、キャラ画像アップロードまで入れています。

Core Loop 作る → 戦う → 覚える → 関係が変わる
AI Mode サーバー接続待ち

この試作でできること

  • 自由入力の行動とチャットカテゴリからバトル演出を生成
  • 友情、因縁、尊敬、警戒の関係値を保存
  • 前半 3 ターン、インターバル、後半 3 ターンの構成
  • 戦後イベントと記憶ログで因縁や戦友関係を残す
  • キャラ画像、詳細設定、備考をローカル保存

Command R+ 設定

公開時の API キーはブラウザではなく、サーバーの環境変数 `COHERE_API_KEY` に置いてください。
